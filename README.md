# CS Tycoon (客服物語) 🎮

一款關於客服日常的開羅風格（Kairo-like）模擬經營遊戲，包含團隊動態、壓力管理以及應對「奧客」Boss 戰。

## 🛠️ 專案狀態
**階段：** 規劃與初步設定

## 📂 資料夾結構
- `assets/`: 存放圖片、音效等外部素材。
- `scenes/`: 存放 Godot 場景檔案 (.tscn)。
- `scripts/`: 存放 GDScript 邏輯代碼 (.gd)。
- `resources/`: 存放自定義的 Godot 資源 (.tres)。
