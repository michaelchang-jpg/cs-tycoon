# CS Tycoon (客服物語) 🎮

A Kairo-like simulation game about customer service life, featuring team dynamics, stress management, and "Karen" bosses.

## 🛠️ Project Status
**Phase:** Planning & Initial Setup

## 📂 Structure
- `assets/`: Textures, audio, and other external assets.
- `scenes/`: Godot scene files (.tscn).
- `scripts/`: GDScript files (.gd).
- `resources/`: Custom Godot resources (.tres).
