# CS Tycoon 夜間驗收清單（5 分鐘）

## 啟動
1. 用 Godot 開 `cs-tycoon`。
2. 執行 `main.tscn`。

## 驗收項目
- [ ] **邊界正常**：員工只在辦公區內走動，不會跑出地板可視範圍。
- [ ] **辦公桌互動**：員工會自動走到辦公桌，進入「工作中」。
- [ ] **飲水機互動**：壓力高時，員工會走去飲水機，進入「喝水中」。
- [ ] **壓力值變化**：
  - 工作中：壓力值上升
  - 喝水中：壓力值下降
- [ ] **UI 顯示**：`目前行為` 與 `壓力值` 會即時更新。

## 你應該看到的畫面
- 左上角資訊板有四行：
  1. 公司資產
  2. 今日產能
  3. 壓力值
  4. 目前行為

## 若有異常
請回報：
1. 畫面截圖
2. 當下 `目前行為` 文字
3. 是否卡在某個點不動
